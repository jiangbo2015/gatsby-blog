### 首先进入命令行模式（ESC 或者 ctrl+m)

操作单元格

-   a: 在上方插入新的 cell
-   b: 在方插入新的 cell

单元格模式切换

-   m: 转换为 markdown
-   y: 转换为 code
-   r: 转换为文本(raw)

运行相关

-   shift + enter: 运行当前单元格，并切换到一个
-   ctrl + enter: 运行当前单元格
-   enter: 进入编辑模式
